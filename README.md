# invoice-qrcode

Frontend App help you scan QRcode on invoice

Demo: https://hcwg.github.io/invoice-qrcode/

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
